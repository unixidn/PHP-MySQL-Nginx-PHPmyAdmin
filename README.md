# PHP MySQL Nginx PHPmyAdmin

#### For Laravel:

Step 1 
````
git clone https://github.com/laravel/laravel.git ./web
````
````
cp ./web/.env.example ./web/.env
````
Step 2
````
make laravel
````

#### Setup

````
make init
````

#### Run Docker Container For Develop

````
make up
````

#### Stop Docker Container For Develop

````
make down
````
Host: 
http://localhost:8000

PHPmyAdmin:
http://localhost:8001
